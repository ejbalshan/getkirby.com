## Example

```php
<?php
echo $page->hasPrevUnlisted(); //will echo 1 if true, nothing if false.
?>
```

### With collection as argument

```php
<?php
$collection = $page->siblings(false)->unlisted()->sortBy('date', 'desc');
echo $page->hasPrevUnlisted($collection); //will echo 1 if true, nothing if false.
?>
```
