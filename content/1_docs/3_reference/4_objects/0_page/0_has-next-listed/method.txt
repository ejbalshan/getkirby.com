## Example

```php
<?php
echo $page->hasNextListed(); //will echo 1 if true, nothing if false.
?>
```

### With collection as argument

```php
<?php
$collection = $page->siblings(false)->listed()->sortBy('date', 'desc');
echo $page->hasNextListed($collection); //will echo 1 if true, nothing if false.
?>
```
